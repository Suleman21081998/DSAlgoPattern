Graph Declaration and Initialization for Competitive Programming

Definition
- A graph is a collection of vertices (nodes) connected by edges.
- Graphs are used to represent networks, routes, dependencies, and relationships.
- Common types:
  - Undirected graph
  - Directed graph
  - Weighted graph
  - Unweighted graph

Important Idea: Internal Working
- A graph stores nodes and edges between them.
- This can be represented using:
  - Adjacency matrix: a 2D array where matrix[u][v] tells whether there is an edge.
  - Adjacency list: each node stores a list of connected nodes.
- Adjacency list is more commonly used in competitive programming because it is memory efficient.

C++
- Declaration using adjacency list:
  vector<vector<int>> graph(n);
- Initialization:
  int n = 5;
  vector<vector<int>> graph(n);
  graph[0].push_back(1);
  graph[0].push_back(2);
  graph[1].push_back(3);

- Basic operations:
  - addEdge(u, v)
  - removeEdge(u, v)
  - traverse neighbors of a node
  - BFS / DFS

- Time complexity:
  - Add edge: O(1)
  - Remove edge: O(degree of node) in adjacency list, O(1) in adjacency matrix
  - Access neighbors: O(degree of node)

- Example:
  vector<vector<int>> graph(4);
  graph[0].push_back(1);
  graph[1].push_back(2);

Java
- Declaration using adjacency list:
  List<List<Integer>> graph = new ArrayList<>();
- Initialization:
  int n = 5;
  List<List<Integer>> graph = new ArrayList<>();
  for (int i = 0; i < n; i++) {
      graph.add(new ArrayList<>());
  }
  graph.get(0).add(1);
  graph.get(0).add(2);

- Basic operations:
  - addEdge(u, v)
  - removeEdge(u, v)
  - get neighbors of a node
  - BFS / DFS

- Time complexity:
  - Add edge: O(1)
  - Remove edge: O(degree of node) in adjacency list
  - Access neighbors: O(degree of node)

- Example:
  List<List<Integer>> graph = new ArrayList<>();
  for (int i = 0; i < 4; i++) graph.add(new ArrayList<>());
  graph.get(0).add(1);

Python
- Declaration using adjacency list:
  graph = [[] for _ in range(n)]
- Initialization:
  n = 5
  graph = [[] for _ in range(n)]
  graph[0].append(1)
  graph[0].append(2)
  graph[1].append(3)

- Basic operations:
  - add_edge(u, v)
  - remove_edge(u, v)
  - get neighbors of a node
  - BFS / DFS

- Time complexity:
  - Add edge: O(1)
  - Remove edge: O(degree of node)
  - Access neighbors: O(degree of node)

- Example:
  graph = [[] for _ in range(4)]
  graph[0].append(1)

Competitive Programming Tips
- Use adjacency list for most graph problems.
- Use BFS for shortest path in unweighted graphs.
- Use DFS for connected components, cycle detection, and path exploration.
- For weighted graphs, use Dijkstra or Bellman-Ford depending on the problem.
