Disjoint Set / Union-Find Declaration and Initialization for Competitive Programming

Definition
- A Disjoint Set Union (DSU), also called Union-Find, is a data structure used to manage a partition of elements into disjoint sets.
- It supports two main operations:
  - union(a, b): merge the sets containing a and b
  - find(x): find the representative/root of the set containing x
- It is commonly used for connected components, cycle detection, Kruskal's algorithm, and network connectivity problems.

Important Idea: Internal Working
- Each element has a parent pointer.
- Initially, every element is its own parent.
- find(x) follows parent links until it reaches the root.
- union(a, b) attaches one root under another root.
- Path compression and union by rank/size make operations very fast.

- Time complexity:

  - Access/Check connectivity: amortized O(1)

- Proper class example in C++:
  class DSU {
  private:
      vector<int> parent, rank;
  public:
      DSU(int n) {
          parent.resize(n);
          rank.assign(n, 0);
          for (int i = 0; i < n; i++) parent[i] = i;
      }

      int find(int x) {
          if (parent[x] != x) {
              parent[x] = find(parent[x]);
          }
          return parent[x];
      }

      void unionSets(int a, int b) {
          int ra = find(a);
          int rb = find(b);
          if (ra == rb) return;
          if (rank[ra] < rank[rb]) swap(ra, rb);
          parent[rb] = ra;
          if (rank[ra] == rank[rb]) rank[ra]++;
      }

      bool connected(int a, int b) {
          return find(a) == find(b);
      }
  };

- Proper class example in Java:
  class DSU {
      private int[] parent, rank;

      public DSU(int n) {
          parent = new int[n];
          rank = new int[n];
          for (int i = 0; i < n; i++) {
              parent[i] = i;
          }
      }

      public int find(int x) {
          if (parent[x] != x) {
              parent[x] = find(parent[x]);
          }
          return parent[x];
      }

      public void union(int a, int b) {
          int ra = find(a);
          int rb = find(b);
          if (ra == rb) return;
          if (rank[ra] < rank[rb]) {
              int temp = ra;
              ra = rb;
              rb = temp;
          }
          parent[rb] = ra;
          if (rank[ra] == rank[rb]) rank[ra]++;
      }

      public boolean isConnected(int a, int b) {
          return find(a) == find(b);
      }
  }

- Proper class example in Python:
  class DSU:
      def __init__(self, n):
          self.parent = list(range(n))
          self.rank = [0] * n

      def find(self, x):
          if self.parent[x] != x:
              self.parent[x] = self.find(self.parent[x])
          return self.parent[x]

      def union(self, a, b):
          ra = self.find(a)
          rb = self.find(b)
          if ra == rb:
              return
          if self.rank[ra] < self.rank[rb]:
              ra, rb = rb, ra
          self.parent[rb] = ra
          if self.rank[ra] == self.rank[rb]:
              self.rank[ra] += 1

      def connected(self, a, b):
          return self.find(a) == self.find(b)

Step-by-Step Example
- Start with: {0}, {1}, {2}, {3}, {4}
- union(0, 1) => {0, 1}, {2}, {3}, {4}
- union(1, 2) => {0, 1, 2}, {3}, {4}
- union(3, 4) => {0, 1, 2}, {3, 4}
- find(2) returns the representative of {0, 1, 2}
- connected(1, 2) => true
- connected(2, 4) => false

Use Cases
- Detecting cycles in graphs
- Finding connected components in a graph
- Kruskal's algorithm for minimum spanning tree
- Network connectivity and friend-circle problems
- Dynamic connectivity problems in competitive programming

Competitive Programming Tips
- Use DSU when you need to merge groups and test whether two elements are in the same group.
- Path compression and union by rank/size are the key optimizations.
- It is one of the most useful structures for graph-related problems.
