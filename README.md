# DSAlgoPattern

This repository contains pattern and data structure concepts along with example solutions. It is intended to help you learn common algorithmic patterns, practice data structure problems, and explore working code for each concept.

## C++ examples

Use the helper script below to compile and run C++ files in this workspace.

From the repository root:

```bash
cd Algorithms/pattern
./run_cpp.sh PrefixSumArray/PrefixSum.cpp
```

Run all C++ examples under `Algorithms/pattern`:

```bash
cd Algorithms/pattern
./run_cpp.sh
```

If you want to run a specific C++ file in another folder, compile and execute it directly. For example:

```bash
cd Algorithms/pattern/TwoPointer/c++
g++ -std=c++17 mostwater.cpp -o mostwater
./mostwater
```

## Java examples

Run the Java prefix sum example from its folder:

```bash
cd Algorithms/pattern/PrefixSumArray/Java
javac PrefixSum.java
java PrefixSum
```
